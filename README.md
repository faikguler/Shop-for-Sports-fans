## Installation
`git clone`
`npm install`

## Usage
The intended use of this website is to provide a place for users to search and find items they want related their sport. 
## Features
Features include: 
- Search bar
- Product image
- Product price
- Active cart count
- Display of current user
- Navigation bar
- Categories
- Responsive design
- Authentication/Sign in
- Multiple page design
